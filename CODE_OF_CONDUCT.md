# Code of Conduct

## Our pledge

We want `creative-department` to be a place where people can discuss craft, contribute skills, and debate design decisions without performing politeness or performing hostility. The bar is simple: treat others like colleagues.

## What that means in practice

**Expected:**
- Direct feedback, specific examples, concrete suggestions
- Disagreement on technical and design decisions, expressed without personal attacks
- Honest "I don't know" / "I was wrong" when that's the honest position
- Patience with newcomers asking questions that have been answered before

**Not acceptable:**
- Personal attacks, slurs, or discriminatory language
- Sustained hostility toward individual contributors
- Deliberate derailing of technical discussions into ideological fights
- Harassment across issues, PRs, or off-platform

## Scope

This code applies to project spaces: issues, pull requests, commit messages, the repository's discussion tabs, and any official communication channel if one is added.

## Reporting

If something crosses the line, email the maintainer at the address on the maintainer's GitHub profile. Reports will be handled privately and promptly.

Enforcement ranges from a private word to a temporary ban to a permanent ban, calibrated to the severity and pattern of behavior.

## Attribution

This code of conduct is written for this project specifically. It draws on the spirit of the [Contributor Covenant](https://www.contributor-covenant.org/) without adopting its full text, because it felt more useful to write it directly than to import.
