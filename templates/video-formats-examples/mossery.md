# Mossery shape — single-take face-cam

**Format slug:** `mossery`
**Length:** 45–90s
**Aspect:** 9:16 vertical

## Structure

Single take. Desk-cam or chest-up frame. One thought, finished. Talking-head face-cam, eyes roughly at the lens. **No teleprompter visible** — if you can see the operator reading, the format breaks.

The hook is one text overlay at the start (0–3s). Nothing else on screen for the rest of the shot.

## What it isn't

- No B-roll
- No music
- No fast cuts (one cut max for a stutter)
- No retention-bait pacing
- No captions stacked on captions (one clean subtitle layer only)
- No on-screen text after the opening hook
- No teleprompter — if the operator can't say it from memory or notes glanced before recording, the script needs to be shorter

## When to use

- Reference-share posts ("there's this thing X is doing that's worth watching")
- Question-register / thinking-aloud ("I keep not closing this question")
- Personal observation when the subject is clear and the receipt is in the saying itself
- Anytime the medium is meant to *perform* what it's describing

## When NOT to use

- Any pitch that requires showing sources / evidence on screen — use split-screen-browser-record instead
- Any pitch where the operator's spoken authority is shaky and a script would help — use explainer instead
- Any pitch with claims that need visual proof (charts, maps, screenshots)

## Reference creators

The shape was named after [@mossery](https://www.instagram.com/mossery/) — single take, single desk, single thought is their consistent move. Other practitioners doing the same shape: priscillaong.co, iggykos, logodecks.

Cross-niche the format also works — chefs, musicians, woodworkers. The format insight is valid even when the topic is far from yours.

## Production cost

**Lowest.** 8–15 min per video (record + minimal edit).

A weekly cadence floor of 4 Mossery Reels/week is achievable inside this format alone.

## Recipe

1. Write 100–150 words for the spoken part (target ~55s at conversational pace)
2. One sentence for the opening hook (text overlay 0–3s)
3. Sit down, glance at the script, hit record
4. Take 1 — speak the whole thing in one go
5. If you stumble, restart at the START of the sentence (not mid-word) — auto-cut-takes will detect the repeat and remove the bad attempt
6. Edit: volume boost 3-4× if phone mic, no music, no music, hook overlay on
7. Ship

## Voice rules for the spoken part

- Connective tissue ("so", "basically", "because", "which means")
- No triplets / staccato lists / aphorism closers
- No em-dash apposition where a period would breathe
- First person OK but not first-person leading
- Trail-off close when the question stays open

## Visual production

- Camera: phone at chin height, lens at eye line
- Lighting: window-light from side preferred; ring-light if dark
- Background: visible but slightly out of focus (real Portrait mode OR post-process via video-pipeline)
- Outfit: consistent across uploads if possible — shape-recognition compounds
