# Video Formats — Worked Examples

Three production-tested vertical video formats. Use these as starting points when filling in your own brand's `VIDEO-FORMATS.md` (see `../VIDEO-FORMATS-template.md`).

These three cover the most common talking-head + screen-record + explainer use cases. Pick all three, pick one, or use them as a reference set for designing your own.

## The three

1. **`mossery.md`** — single-take face-cam, conversational, no teleprompter, no B-roll. Lowest production cost. Best for thinking-aloud / reference-share / observations.
2. **`split-screen-browser-record.md`** — face-cam on top + screen recording of browser tabs/sources on bottom. Best for "10 outlets covered this, none read it this way" cultural observations.
3. **`explainer.md`** — VO + B-roll/maps/motion graphics, face optional and minimal. Johnny-Harris / Nolan-Perkins / Cleo-Abram shape. Highest cost, highest production value.

## Why named formats matter

- **Cost prediction** — you know "this is a 15-min mossery" vs "this is a 4-hour explainer" before recording
- **Pitcher proposes a format** — every video pitch declares its slug, no "what should this be?" stalling
- **Audience recognition** — shape-consistency across uploads compounds
