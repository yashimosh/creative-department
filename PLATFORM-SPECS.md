# Platform Specifications Reference

Centralized, maintained reference for all digital platform specs. All creative and caption skills reference this file.

_Last updated: 2026-04-07_

---

## Instagram

### Feed Post
- **Dimensions:** 1080x1080 (square), 1080x1350 (portrait, recommended), 1080x566 (landscape)
- **Aspect ratio:** 1:1, 4:5, 1.91:1
- **File types:** JPG, PNG
- **Max file size:** 30MB
- **Caption:** 2,200 characters max
- **Hashtags:** 30 max (13 recommended)

### Carousel
- **Slides:** 2–20
- **Dimensions:** 1080x1080 or 1080x1350 (must be consistent across slides)
- **File types:** JPG, PNG, MP4 per slide
- **Max file size:** 30MB per image, 4GB per video slide

### Story
- **Dimensions:** 1080x1920
- **Aspect ratio:** 9:16
- **Duration:** up to 60s per story
- **Safe zone:** Keep text/CTAs within center 1080x1420 (avoid top 250px for username, bottom 250px for swipe-up/reply)
- **File types:** JPG, PNG, MP4

### Reel
- **Dimensions:** 1080x1920
- **Aspect ratio:** 9:16
- **Duration:** up to 15 min
- **Recommended duration:** 15–30s for engagement
- **Cover image:** 1080x1920, safe zone center 1080x1420
- **File types:** MP4
- **Max file size:** 4GB
- **Caption:** 2,200 characters

### Profile Picture
- **Dimensions:** 320x320 (displayed as 110x110 circle)

## LinkedIn

### Feed Post (Image)
- **Dimensions:** 1200x1200 (square), 1200x628 (landscape), 1080x1350 (portrait)
- **Recommended:** 1200x1200 or 1080x1350
- **File types:** JPG, PNG, GIF
- **Max file size:** 10MB
- **Caption:** 3,000 characters (first 210 visible before "see more")

### Carousel (Document Post)
- **Dimensions:** 1080x1080 or 1080x1350 (PDF pages)
- **File types:** PDF
- **Max pages:** 300 (10–15 recommended)
- **Max file size:** 100MB

### Video
- **Dimensions:** 1920x1080 (landscape), 1080x1920 (portrait), 1080x1080 (square)
- **Duration:** 3s–10 min
- **File types:** MP4
- **Max file size:** 5GB

### Article Cover Image
- **Dimensions:** 1920x1080

### Company Page Banner
- **Dimensions:** 1128x191

## X / Twitter

### Post Image
- **Dimensions:** 1200x675 (landscape, 16:9), 1080x1080 (square)
- **In-feed display:** cropped to ~2:1 in timeline
- **File types:** JPG, PNG, GIF
- **Max file size:** 5MB (images), 15MB (GIF)
- **Max images:** 4 per post

### Post Copy
- **Characters:** 280 (X Premium: 25,000)
- **Hashtags:** 1–2 max

### Video
- **Dimensions:** 1920x1080 (landscape), 1080x1920 (portrait)
- **Duration:** 0.5s–140s (up to 10 min for some accounts)
- **File types:** MP4
- **Max file size:** 512MB

### Profile Picture
- **Dimensions:** 400x400

### Header Image
- **Dimensions:** 1500x500

## YouTube

### Video
- **Dimensions:** 1920x1080 (1080p), 3840x2160 (4K)
- **Aspect ratio:** 16:9
- **File types:** MP4, MOV, AVI, WMV
- **Max file size:** 256GB
- **Max duration:** 12 hours

### Thumbnail
- **Dimensions:** 1280x720
- **Min width:** 640px
- **File types:** JPG, PNG, GIF, BMP
- **Max file size:** 2MB
- **Safe zone:** Avoid bottom-right 200x40px (timestamp overlay)

### Shorts
- **Dimensions:** 1080x1920
- **Aspect ratio:** 9:16
- **Duration:** up to 3 min
- **Recommended:** 15–60s

### Channel Banner
- **Dimensions:** 2560x1440
- **Safe zone for all devices:** center 1546x423

### Channel Profile Picture
- **Dimensions:** 800x800

## TikTok

### Video
- **Dimensions:** 1080x1920
- **Aspect ratio:** 9:16
- **Duration:** 3s–10 min
- **Recommended duration:** 15–60s
- **File types:** MP4, MOV
- **Max file size:** 287.6MB (mobile), 10GB (desktop)

### Safe Zone
- Keep critical content within center 750x1260 area:

  - Bottom: 270px reserved (CTA buttons, description)
  - Right: 100px reserved (engagement buttons)

- **Caption:** 4,000 characters max
- **Hashtags:** 3–5 recommended

## Email

### Email Body
- **Max width:** 600px (for compatibility)
- **Header image:** 600x200 or 600x300
- **Hero image:** 600x400
- **File types:** JPG, PNG, GIF (embedded)
- **Total email size:** under 100KB HTML, images hosted externally

## General Notes

- Always export at 2x resolution for retina displays when the platform supports it
- sRGB color space for all digital assets
- CMYK only for print (see `/review-print` for print specs)
- Video codec: H.264 for maximum compatibility, H.265 for quality where supported
- Audio: AAC, 128kbps minimum, 48kHz sample rate
- Always test safe zones — platform UI overlays change frequently
